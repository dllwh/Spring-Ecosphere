# 具备的功能(v0.1)

- Spring Boot (后端)

  - 通过在Spring Boot中建立基于RestFul-API并使用**@RequestMapping**实现一个基本的CRUD逻辑
  - 处理CORS(跨域资源共享)
  - 支持热加载
	
- VueJS & Vue-Router (前端)
	- 遵循ECMAScript 6 规范
	- 如何和后端进行数据交互
